Instructions

Make some art! Have the angleIt function return a number and it will be applied to all squares.

How it works:

  • angleIt(row, col, time) is called on each square
  • row and col are the square's position (0-9)
  • time is the elapsed times (in frames)
  • The function should return an angle in degrees.

Edit the function on the right and click Apply Function to see your changes!

Try cutting-and-pasting this into the function editor, and uncomment/comment different lines to see the results: 
function angleIt(row, col, time) {
    return 57; 
    // return row * time; 
    // return col + row;
    // return time % (row + 1);
}
function angleIt(row, col, time) {
    if ((row + col) % 2 === 1) {
        return 30;
    } else {
        return 0;
    }
}
